# 画面仕様書一覧

## 画面一覧

| 画面名 | パス | 機能概要 |
|--------|------|----------|
| チャート表示画面 | `/` | 株価ローソク足チャートとニュース情報を統合表示するメイン画面 |
| 株一覧画面 | `/stocks` | 登録済み株銘柄の一覧表示と検索、管理画面 |
| 株登録画面 | `/stocks/new` | 新規株銘柄を登録するフォーム画面 |
| 株価チャート画面 | `/chart/:stockCode` | 特定銘柄の株価データをチャート表示する画面 |
| 株価インポート画面 | `/import` | 株価データをCSVファイルからインポートする画面 |
| ニュースインポート画面 | `/news-import` | ニュースデータをCSVファイルからインポートする画面 |
